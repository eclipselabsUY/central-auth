# eclipse-labs-auth
 Centralized authentication for all services supported by Eclipse Labs

## Documentation

Project documentation lives in `docs/` and is configured with MkDocs via `mkdocs.yml`.

### Run docs locally

1. Install MkDocs:
   - `pip install mkdocs`
2. Start local docs server from repo root:
   - `mkdocs serve`
3. Open:
   - `http://127.0.0.1:8000`

### Build static docs

- `mkdocs build`
